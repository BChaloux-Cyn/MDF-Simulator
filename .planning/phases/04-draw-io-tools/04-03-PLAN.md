ruamel.yaml round-trip pattern (from RESEARCH.md, verified against 0.19.1):

```python
from ruamel.yaml import YAML
import io

def _read_yaml_roundtrip(path: Path):
    yaml_rt = YAML()
    data = yaml_rt.load(path.read_text())   # returns CommentedMap (not plain dict)
    return data

def _write_yaml_roundtrip(data, path: Path):
    yaml_rt = YAML()
    yaml_rt.default_flow_style = False
    buf = io.StringIO()
    yaml_rt.dump(data, buf)
    path.write_text(buf.getvalue(), encoding="utf-8")
# WARNING: isinstance(data, dict) returns False for CommentedMap — use data["key"] directly
# Pydantic model_validate() handles CommentedMap correctly
```

**validate_drawio(domain: str, xml: str) -> list[dict]:**

Algorithm:
1. Parse xml with defusedxml.ElementTree.fromstring(xml.encode("utf-8")); on exception return [_make_issue("XML parse error: {exc}", "xml_input", severity="error")]
2. Build valid_styles = set(BIJECTION_TABLE.values())
3. For each mxCell in root.iter("mxCell"):
   - Skip cells with id in ("0", "1") — base infrastructure cells
   - Skip cells with no "style" attribute
   - Skip cells whose style IS in valid_styles
   - If style not in valid_styles: append _make_issue(f"Unrecognized Draw.io cell style", location=f"xml:cell:{cell.get('id','?')}", value=cell.get("style"), fix="Use only canonical MDF style strings from BIJECTION_TABLE", severity="error")
4. Return issues list (may be empty)

Note: style strings in real Draw.io XML may have trailing semicolons or minor variations. Use exact string match against BIJECTION_TABLE.values() first. If no exact match, also try stripping trailing ";" — if match found after strip, it is still considered valid (Draw.io sometimes adds a trailing semicolon). If still no match, it is unrecognized.

**sync_from_drawio(domain: str, xml: str) -> list[dict]:**

Algorithm:
1. Validate XML first: call _parse_xml_safe(xml) using defusedxml; return early on parse error
2. Load domain_path = MODEL_ROOT / domain; check exists
3. Load class-diagram.yaml with yaml.safe_load + ClassDiagramFile.model_validate; return issues on failure
4. For each mxCell in root.iter("mxCell"):
   - Skip base cells (id "0", "1")
   - Skip cells with no canonical ID (no ":" in id, or id is None)
   - Parse element type from ID: split on ":" → [domain_part, type_part, ...]
   - type_part "class", "assoc", "state", "trans" are the sync-able types

   - type_part "state" where state_name == "__initial__": skip (pseudo)

   - RENAMED: not detected (IDs are canonical — a renamed class would have a new ID; treat as new+deleted)
   - NEW assoc cells: create new Association with name from label, point_1/point_2 from source/target cell IDs parsed, mult placeholders "?" (Claude's discretion)

**HTML tag stripping for transition labels:**
```python
import re
def _strip_html(text: str) -> str:
    return re.sub(r'<[^>]+>', '', text).replace('&lt;', '<').replace('&gt;', '>').replace('&amp;', '&')
```

**Parsing transition label lines:**
```python
def _parse_trans_label(label: str) -> tuple[str, str | None]:
    """Return (event_name, guard_or_None) from a multi-line transition label."""
    lines = [_strip_html(l).strip() for l in label.split('<br>') if _strip_html(l).strip()]
    # line 0 is canonical ID (skip); line 1 is event sig; line 2 optionally guard
    event_line = lines[1] if len(lines) > 1 else lines[0]
    event_name = event_line.split('(')[0].strip()
    guard = None
    for line in lines[2:]:
        if line.startswith('[') and line.endswith(']'):
            guard = line[1:-1]
    return event_name, guard
```

```python
# ---------------------------------------------------------------------------
# Draw.io tools (MCP-05 / MCP-06 / MCP-07)
# ---------------------------------------------------------------------------

@mcp.tool()
def render_to_drawio_tool(domain: str) -> list[dict]:
    """
    Generate all Draw.io diagrams for a domain from YAML.

    Writes class-diagram.drawio and one state diagram per active class.
    Skip-if-unchanged: if structure matches existing .drawio, preserves layout positions.
    Returns per-file list of {"file": path, "status": "written"|"skipped"} dicts,
    plus any error issue dicts.
    Never raises — all errors returned as structured data.
    """
    return render_to_drawio(domain)


@mcp.tool()
def render_to_drawio_class_tool(domain: str) -> list[dict]:
    """
    Generate class-diagram.drawio for a domain from class-diagram.yaml.

    Skip-if-unchanged: preserves engineer layout if YAML structure is unchanged.
    Returns [{"file": path, "status": "written"|"skipped"}] plus any error issues.
    Never raises.
    """
    return render_to_drawio_class(domain)


@mcp.tool()
def render_to_drawio_state_tool(domain: str, class_name: str) -> list[dict]:
    """
    Generate state-diagrams/<ClassName>.drawio for one active class.

    Skip-if-unchanged: preserves engineer layout if state machine structure is unchanged.
    Returns [{"file": path, "status": "written"|"skipped"}] plus any error issues.
    Never raises.
    """
    return render_to_drawio_state(domain, class_name)


@mcp.tool()
def validate_drawio_tool(domain: str, xml: str) -> list[dict]:
    """
    Validate Draw.io XML against the canonical MDF bijection schema.

    Returns empty list for valid canonical XML.
    Returns error issues for any mxCell with an unrecognized style string.
    Call this before sync_from_drawio to catch freeform edits.
    Never raises.
    """
    return validate_drawio(domain, xml)


@mcp.tool()
def sync_from_drawio_tool(domain: str, xml: str) -> list[dict]:
    """
    Sync Draw.io XML topology changes back to YAML model files.

    Merges structural changes (add/remove/rename states, transitions, classes,
    associations) while preserving YAML-only fields (pycca action bodies, guards,
    attribute types, method signatures).
    Automatically runs validate_model after sync; its issues appear in the returned list.
    Returns info-severity issues for each deleted element (pycca bodies may be lost).
    Never raises.
    """
    return sync_from_drawio(domain, xml)
```
