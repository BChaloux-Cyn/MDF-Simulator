## Changes by File

**engine/manifest.py** — `transition_table` type annotation updated to `dict[tuple[str, str], list[TransitionEntry]]`.

**compiler/manifest_builder.py** — `_build_transition_table` return type updated; `table[key] = entry` replaced with `table.setdefault(key, []).append(entry)` to accumulate guard siblings without key collision.

**compiler/codegen.py:**
- `import warnings` added at module level
- `used_types` filter loop replaced with `for type_name in sorted(type_registry.keys())` — unconditional emission
- `except Exception:` replaced with `except Exception as exc: warnings.warn(...)` in both action and guard compile paths
- `action_fn_map` wiring loop updated to iterate list entries per key
- `guard_fn_map` loop updated with `enumerate` and sibling index naming (`guard_S_E_N` for idx > 0)
- `_render_transition_table` updated to emit list literals per key

**engine/scheduler.py — `_process_event`:** Single-entry lookup replaced with guard chain: iterates `entries` list, picks first candidate whose guard returns True, or first unguarded entry as fallback. `_process_creation` updated to use `_creation_entries[0]` pattern.

**engine/bundle_loader.py:** Rebinding loop updated to iterate `live_entries` list and match to `manifest_entries` by index (same deterministic ordering guaranteed by codegen D-07).
