<objective>
Build the canonical-based loader and manifest builder. The compiler's primary input format is the canonical JSON already used for YAML↔Draw.io translation (CanonicalClassDiagram / CanonicalStateDiagram from schema/drawio_canonical.py). This plan:

1. Extracts the YAML→canonical conversion logic from tools/drawio.py into schema/canonical_builder.py so compiler/ and tools/ share one path.
2. Builds compiler/loader.py — walks model root, converts YAML to canonical objects, returns LoadedModel.
3. Builds compiler/manifest_builder.py — takes canonical objects, produces DomainManifest with D-03/D-04/D-07/D-13/D-14 rules.
4. Builds compiler/senescence.py — static senescent state classifier.

Purpose: Produce the fully-populated DomainManifest that Plan 04 will serialize and Plan 05 will validate against the elevator model.
</objective>

Correct encoding: **`can't_happen = absent cell`** (omit from `transition_table` dict), **`event_ignored = present cell with all-None fields`** (`next_state=None, action_fn=None, guard_fn=None`).
