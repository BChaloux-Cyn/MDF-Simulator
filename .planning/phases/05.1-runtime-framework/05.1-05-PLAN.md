```python
from typing import Any, Generator
from engine.registry import InstanceRegistry
from engine.relationship import RelationshipStore
from engine.scheduler import ThreeQueueScheduler
from engine.clock import SimulationClock
from engine.bridge import BridgeMockRegistry
from engine.event import Event
from engine.microstep import MicroStep


class SimulationContext:
    """The single API surface that generated action code calls into (D-35).
    
    Owns: scheduler, instance registry, relationship store, bridge mock
    registry, clock. All methods per D-36.
    """
    
    def __init__(self, domain_manifest: dict, bridge_mocks: dict | None = None):
        class_defs = domain_manifest["class_defs"]
        associations = domain_manifest.get("associations", {})
        generalizations = domain_manifest.get("generalizations", {})
        
        self.clock = SimulationClock()
        self.registry = InstanceRegistry(class_defs)
        self.relationships = RelationshipStore(associations)
        self.bridge_registry = BridgeMockRegistry(mocks=bridge_mocks)
        self.scheduler = ThreeQueueScheduler(
            registry=self.registry,
            clock=self.clock,
            class_defs=class_defs,
            generalizations=generalizations,
        )
        self._running = False
```

Methods per D-36 — each delegates to the appropriate component and handles micro-step collection:

- `generate(event_type: str, sender_class: str, sender_id: dict, target_class: str, target_id: dict, args: dict | None = None, delay_ms: float | None = None) -> list[MicroStep]`
  Creates Event and calls `scheduler.enqueue()`. Returns micro-steps.

- `cancel(event_type: str, sender_id: dict, target_id: dict) -> list[MicroStep]`
  Delegates to `scheduler.cancel()`.

- `create_sync(class_name: str, identifier: dict, initial_state: str, attrs: dict | None = None) -> list[MicroStep]`
  Delegates to `registry.create_sync()`.

- `create_async(class_name: str, identifier: dict, initial_state: str, attrs: dict | None = None) -> list[MicroStep]`
  Calls `registry.create_async()`, enqueues the returned creation Event via `scheduler.enqueue()`. Returns combined micro-steps.

- `delete(class_name: str, identifier: dict, sync: bool = False) -> list[MicroStep]`
  If sync: delegates to `registry.delete_sync()`. If async: calls `registry.delete_async()`, enqueues returned Event. Returns micro-steps.

- `relate(rel_id: str, class_a: str, id_a: dict, class_b: str, id_b: dict) -> list[MicroStep]`
  Delegates to `relationships.relate()`.

- `unrelate(rel_id: str, class_a: str, id_a: dict, class_b: str, id_b: dict) -> list[MicroStep]`
  Delegates to `relationships.unrelate()`.

- `navigate(rel_id: str, from_class: str, from_id: dict) -> list[dict]`
  Delegates to `relationships.navigate()`.

- `select_any(class_name: str, where: Callable | None = None) -> dict | None`
  Calls `registry.lookup_all(class_name)`, applies optional where filter, returns first match or None.

- `select_many(class_name: str, where: Callable | None = None) -> list[dict]`
  Calls `registry.lookup_all(class_name)`, applies optional where filter, returns all matches.

- `bridge(operation: str, args: dict | None = None) -> tuple[Any, list[MicroStep]]`
  Delegates to `bridge_registry.call()`. Returns (return_value, [BridgeCalled micro-step]).

- `now() -> float`
  Returns `clock.now()`.

- `load_scenario(scenario: dict) -> list[MicroStep]`
  Process scenario dict with keys:
  - `instances`: list of `{"class": str, "identifier": dict, "initial_state": str, "attrs": dict}` — create each via `create_sync()`.
  - `events`: list of `{"class": str, "instance": dict, "event": str, "args": dict}` — enqueue each.
  Returns all micro-steps from setup.

- `execute() -> Generator[MicroStep, None, None]`
  Sets `self._running = True`. Yields from `scheduler.execute()`. Sets `self._running = False` when generator exhausts.

Module-level function:

```python
def run_simulation(
    domain_manifest: dict,
    scenario: dict | None = None,
    bridge_mocks: dict | None = None,
) -> Generator[MicroStep, None, None]:
    """Top-level entry point for running a simulation.
    
    Creates a SimulationContext, loads the scenario, and yields
    all micro-steps from execution.
    """
    ctx = SimulationContext(domain_manifest, bridge_mocks)
    if scenario:
        setup_steps = ctx.load_scenario(scenario)
        yield from setup_steps
    yield from ctx.execute()
```
