```python
def advance_to(self, ms: float) -> None:
    """Jump simulation time forward to ``ms`` (D-15).

    - Sets ``self._time_ms = max(self._time_ms, ms)`` in both modes, so
      time never regresses.
    - Real-time mode (``fast_forward=False``): sleeps
      ``(ms - self._time_ms) / (speed_multiplier * 1000)`` real seconds
      before setting the time. With default ``speed_multiplier=1000.0``,
      a 10,000 ms model delay takes 10 ms of real time.
    - Fast-forward mode (``fast_forward=True``): no sleep, no scaling — the
      clock jumps immediately.
    - No-op if paused (D-33 existing semantics preserved).
    """
    if self._paused:
        return
    if not self._fast_forward:
        delta = ms - self._time_ms
        if delta > 0:
            time.sleep(delta / (self._speed_multiplier * 1000.0))
    self._time_ms = max(self._time_ms, ms)
```
