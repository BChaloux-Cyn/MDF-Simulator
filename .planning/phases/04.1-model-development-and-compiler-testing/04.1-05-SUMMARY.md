## Decisions Made

- `select any ind related by elev->R12` is valid grammar because `select_related_stmt` uses `NAME "->" NAME` (not `"self" "->" NAME`), so any variable can be the traversal source.
- YAML `Off` is a boolean keyword in YAML 1.1 (which PyYAML uses by default) and coerces to `False`. All occurrences of `Off` as state names and transition endpoints must be quoted.

- **Fix:** Quoted all occurrences of `Off` as `"Off"` throughout FloorIndicator.yaml
